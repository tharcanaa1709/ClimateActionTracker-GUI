# 🌍 Climate Action Tracker System (GUI)

## SDG 13: Climate Action

This project is a Java GUI-based application developed using Object-Oriented Programming (OOP) principles. It helps users track their daily carbon emissions and encourages environmentally friendly behavior.

## 🔑 Features
- Login System
- Add Activity (Transport, Electricity, Waste)
- Calculate Carbon Emission
- View Result (Low / Medium / High)
- Get Eco-Friendly Recommendations
- Save Data to File

## 🧠 OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## 💻 Technologies
- Java
- Swing (GUI)
- File Handling

## 🚀 How to Run
```bash
javac *.java
java MainApp
